## Publication

1. Mettre `index.html`, `.nojekyll`, `README.md` et le dossier `fichiers/` à la racine du repo `portfolio-t`.
2. Dans GitHub : Settings > Pages.
3. Source : Deploy from a branch.
4. Branch : main.
5. Folder : / (root).

URL attendue : https://toxatiger.github.io/portfolio-t/
